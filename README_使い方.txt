家事記録 Web版 v2

■ 仕様
・iPhone/iPad向け
・白背景 / 黒文字
・西暦、月、前半/後半を選択
・曜日を自動入力
・表の各マスをタップして入力
・風呂掃除：セルをタップするだけで、空欄 → 15分 → 30分 → ... → 120分 → 空欄
・炊飯：○ / 未記入
・ゴミ集め：○ / 未記入
・掃除機：○ / 未記入
・おつかい：上段メモ、下段金額
・+α：上段メモ、下段15分単位、最大180分
・入力途中データは端末内に自動保存
・集計ボタンで開閉
・集計結果は上スワイプで開く / 下スワイプで閉じる
・おつかい使用と合計表示
・PDFを作成ボタンで印刷画面を開く
・PWA対応

■ v2 修正内容
・右端がギリギリ見切れないように表の左右余白を調整
・ページ全体ではなく表部分だけをスクロールする構造に変更
・西暦、前半/後半、期間、項目ヘッダー、集計ボタン、PDF作成ボタンを常に表示
・風呂掃除はセルタップだけで15分単位入力できるように変更

■ 注意
Web版では、Swift版と完全に同じ「iCloudへ直接PDF保存」はできません。
PDFを作成ボタンを押したあと、iPad/iPhoneの印刷・共有画面から「ファイルに保存」を選んでiCloud Driveへ保存してください。

■ ホーム画面に追加
HTTPSで公開したURLをSafariで開き、
共有ボタン → ホーム画面に追加
を選んでください。

■ データ保存
入力途中データはブラウザのlocalStorageに保存されます。
Safariの履歴・Webサイトデータを削除すると消える可能性があります。
必要に応じて「データ書き出し」でバックアップしてください。


■ v3 修正内容
・常に表示する部分を「期間」「項目タイトル」「集計/PDF作成ボタン」に変更
・前半/後半選択とその上の部分は、全体スクロールで隠れてもよい仕様に変更
・表の中をスクロールした場合は表だけスクロール
・表の外をスクロールした場合は全体スクロール
・風呂掃除の入力済みセルを2秒以上長押しすると未記入にリセット
・風呂掃除セルの通常タップは、空欄 → 15分 → 30分 → ... → 120分 → 空欄 のまま
・更新反映を安定させるため、Service Workerをネットワーク優先に変更


■ v4 修正内容
・ダブルタップ時の画面拡大を抑制
・風呂掃除の入力済みマスを1.6秒以上長押しで未記入にリセットへ変更
・長押し削除後、指を離したときに15分が再入力される誤動作を防止
・表の範囲外をスクロールした場合でも、表側の縦スクロールも同期して進むように変更


■ v5 修正内容
・風呂掃除の入力済みマスを1.3秒以上長押しで未記入にリセットへ変更
・集計ボタンとPDF作成ボタンを画面下に完全固定
・前半/後半の選択部分を表の左側へ移動
・前半/後半の選択部分を縦向き表示に変更


■ v6 修正内容
・前半/後半切り替えボタンを表の左側中央付近へ移動
・前半/後半切り替えボタンの縦幅を約2倍に変更
・集計結果を開いているときも、集計/PDF作成ボタンと同じように固定表示
・スクロールして期間表示が画面上部まで来たら固定されるように調整
