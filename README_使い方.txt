家事記録 Web版

■ 仕様
Swift Playgrounds版の仕様に合わせたWebアプリ版です。

・iPhone/iPad向け
・白背景 / 黒文字
・西暦、月、前半/後半を選択
・曜日を自動入力
・表の各マスをタップして入力
・風呂掃除：15分単位、最大120分
・炊飯：○ / 未記入
・ゴミ集め：○ / 未記入
・掃除機：○ / 未記入
・おつかい：上段メモ、下段金額
・+α：上段メモ、下段15分単位、最大180分
・入力途中データは端末内に自動保存
・集計ボタンで開閉
・集計結果は上スワイプで開く / 下スワイプで閉じる
・おつかい使用と合計表示
・PDFを作成ボタンで印刷画面を開く
・PWA対応

■ 注意
Web版では、Swift版と完全に同じ「iCloudへ直接PDF保存」はできません。
PDFを作成ボタンを押したあと、iPad/iPhoneの印刷・共有画面から「ファイルに保存」を選んでiCloud Driveへ保存してください。

■ ホーム画面に追加
HTTPSで公開したURLをSafariで開き、
共有ボタン → ホーム画面に追加
を選んでください。

■ データ保存
入力途中データはブラウザのlocalStorageに保存されます。
Safariの履歴・Webサイトデータを削除すると消える可能性があります。
必要に応じて「データ書き出し」でバックアップしてください。
